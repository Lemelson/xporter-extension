Features

Everything you need, nothing you don't

📊

CSV, JSON, XLSX Export

Export data in clean, structured formats. Open in Excel, Google Sheets, or any data analysis tool.

⚡

Smart Rate Limiting

Built-in intelligent rate limiter prevents API throttling. Automatically pauses and resumes export.

🔄

Resume & Download

Stop mid-export and resume later. Download what you've collected so far at any time.

🌍

14 Languages

Interface available in English, Russian, Spanish, French, German, Japanese, Korean, Chinese, and more.

🎨

Themes & Palettes

Dark and light modes with 6 color palettes. Auto-theme follows your OS preferences.

🔒

100% Private

All processing happens locally in your browser. Zero data sent to external servers. Fully open source.

Export Types

Four export modes

📝

Posts

Text, engagement stats, media URLs, hashtags, language, timestamps

👥

Followers

Name, username, bio, location, follower/following counts, join date

➡️

Following

Full profile data of accounts the user follows

✅

Verified Followers

Blue-verified followers with complete profile information

How It Works

Three steps to your data

1

Install the extension

Add XPorter to Chrome from the Chrome Web Store. No sign-up, no account needed.

2

Visit any X profile

Navigate to any public X (Twitter) profile page. Click the XPorter icon in your toolbar.

3

Choose & export

Select what to export — posts, followers, following, or verified followers. Click export and download your file.

🛡️

Your privacy, guaranteed

XPorter processes everything locally in your browser. No data ever leaves your machine. No analytics, no tracking, no ads. The source code is fully open for audit.
