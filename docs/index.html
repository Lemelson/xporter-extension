Exporting takes four clicks

No setup, no sign-up. Install and go.

Open XPorter

Click the XPorter icon in your toolbar while logged in to X.

Enter a username

Type any profile handle, e.g. elonmusk.

Pick type & format

Posts, followers, following or verified — as CSV, JSON or Excel.

Start exporting

Hit start and download your file. That's it.

Your data stays yours

XPorter runs entirely in your browser using the X session you're already logged into. There are no XPorter servers, no account, and no trackers during use — your exported data goes straight from X into your file and never passes through us. The only thing XPorter ever sends is an anonymous, non-personal usage summary when you uninstall, so churn can be understood and the extension improved.
